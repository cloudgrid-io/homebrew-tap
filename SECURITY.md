# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly. Do not open a public issue.

Email: security@cloudgrid.io

We will acknowledge your report within 48 hours and provide a timeline for
a fix.

## Scope

This repository contains only a Homebrew formula and CI configuration. The
CLI source code lives at [cloudgrid-io/cli](https://github.com/cloudgrid-io/cli).
Report CLI vulnerabilities there.
